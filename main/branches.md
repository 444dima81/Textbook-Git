### Команды для работы с ветками

---

Создание новой ветки и переход в нее `git checkout -b <branch name>`

Отобразить список всех веток `git branch`

Переименовать текущую ветку `git branch -m <new branch name>`

Чтобы удалить ветку `git branch -d <branch name>`

Чтобы слить одну ветку с действующей `git marge <feature branch name>`
